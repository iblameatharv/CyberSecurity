# CyberGuard: Exploring Digital Security

**CyberGuard** is an interactive educational website designed to introduce and explain the fundamentals of **cybersecurity** in an engaging and visually dynamic way. It combines storytelling, interactive modules, and gamified learning elements to help users understand how to stay safe in the digital world.

---

## Overview

CyberGuard serves as a comprehensive guide for learners to explore the world of cybersecurity.  
It blends visual effects, interactivity, and thematic storytelling to explain essential topics such as cyber threats, defenses, ethics, laws, and online safety.  

The website uses a **futuristic cyberpunk design**, representing the digital frontier where technology and awareness intersect.

---

## Features

### 1. Dynamic Hero Section
- Animated introduction with terminal-style effects.  
- Real-time binary rain simulation.  
- Glitch and typewriter animations for immersive engagement.

### 2. Interactive Learning Chapters
- Ten expandable chapters covering major cybersecurity concepts:
  - Cyber Threats
  - Password Protection
  - Cyber Ethics & Laws
  - Social Media Safety
  - Future of Cyber Security  
- Each chapter includes educational text, tips, and small interactive components.

### 3. Video Presentation Mode
- Integrated slide-style presentation system with navigation controls.
- Dynamic progress bar and smooth transition effects.

### 4. Cyber Security Quiz
- Interactive quiz to test users' knowledge.
- Displays live score tracking and results at the end.

### 5. Password Strength Checker
- Real-time analysis of password complexity.
- Color-coded strength meter and feedback text.

### 6. Encryption Demo
- Simple Caesar cipher-based text encryption feature.
- Encourages hands-on understanding of encryption principles.

### 7. Theme Customization
- Five futuristic color themes:
  - Cyberpunk  
  - Matrix  
  - Neon  
  - Ocean  
  - Sunset  
- Each theme applies instantly with animated transitions.

### 8. Copy Protection System
- Right-click and inspection prevention overlay.  
- Alerts users when attempting to copy or inspect code.

## Technologies Used

- **HTML5** for structure  
- **CSS3** (with animations, transitions, and variables) for design  
- **JavaScript (jQuery)** for interactivity  
- **Bootstrap 5** for layout and responsiveness  
- **Font Awesome** for icons  
- **Google Fonts (Orbitron & Roboto)** for typography

## How to Use

1. **Open the project**  
   - Double-click `index.html` or open it through a local server.

2. **Navigate**  
   - Use the navbar or scroll to move between sections.

3. **Learn Interactively**  
   - Expand chapters, take the quiz, and try the encryption and password checker.

4. **Customize**  
   - Use the bottom-right theme switcher to explore different visual themes.
